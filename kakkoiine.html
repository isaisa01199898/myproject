三平方の定理

三平方の定理とは

三平方の定理（ピタゴラスの定理）とは、直角三角形の辺の長さに関する定理です。

直角三角形の斜辺を c、他の二辺を a と b とした場合、次の関係式が成り立ちます：

公式

c² = a² + b²

例

例えば、辺の長さが 3cm と 4cm の直角三角形を考えると、斜辺の長さ c は以下のように計算できます：

c² = 3² + 4² = 9 + 16 = 25

c = √25 = 5

したがって、斜辺の長さは 5cm となります。

応用

  • 建築や測量での距離計算
  • 物理学における力の合成
  • コンピュータグラフィックスでの座標計算
